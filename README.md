## demos

mokjs的所有示例代码

## 目录结构说明

demos/
   ├─ blog/		- JS模块化示例  
   │   ├─ build		- 构建目录  
   │   ├─ src-node	- 遵循CommonJS Modules模块化规范（像Node.js一样）的项目源码  
   │   ├─ src-define    - 遵循CMD模块化规范的项目源码  
   │   └─ static	     - 静态资源（HTML页面、图片等）  
   ├─ css/		  - CSS模块化示例  
   ├─ css-build/	- CSS项目构建目录  
   ├─ html/		 - HTML模块化示例  
   └─ www/		 - HTML项目构建目录
